# Glosario de Servicios AWS

Este glosario proporciona una descripción rápida de los servicios de AWS más comunes incluidos en este diagramador.

| Servicio | Categoría | Descripción |
| :--- | :--- | :--- |
| **AWS Lambda** | Cómputo | Ejecuta código sin necesidad de administrar servidores (Serverless). |
| **Amazon EC2** | Cómputo | Servidores virtuales en la nube para ejecutar aplicaciones. |
| **Amazon S3** | Almacenamiento | Almacenamiento de objetos escalable y seguro para cualquier tipo de datos. |
| **Amazon RDS** | Base de Datos | Servicio de base de datos relacional administrado (MySQL, PostgreSQL, etc.). |
| **Amazon DynamoDB** | Base de Datos | Base de datos NoSQL clave-valor rápida y flexible. |
| **Amazon VPC** | Redes | Red virtual aislada para lanzar recursos de AWS. |
| **AWS CloudFront** | Redes | Red de entrega de contenido (CDN) global rápida y segura. |
| **Amazon API Gateway** | Redes | Servicio para crear, publicar, mantener, monitorear y asegurar APIs. |
| **Elastic Load Balancing** | Redes | Distribuye automáticamente el tráfico de aplicaciones entrante. |
| **Amazon ECS** | Contenedores | Servicio de orquestación de contenedores Docker nativo de AWS. |
| **Amazon EKS** | Contenedores | Servicio de Kubernetes administrado para ejecutar aplicaciones en contenedores. |
| **AWS Fargate** | Cómputo | Motor de computación serverless para contenedores. |
| **Amazon SNS** | Integración | Servicio de mensajería pub/sub totalmente administrado. |
| **Amazon SQS** | Integración | Servicio de colas de mensajes administrado para desacoplar servicios. |
| **AWS Step Functions** | Integración | Coordinación de componentes distribuidos y servicios mediante flujos de trabajo visuales. |
| **Amazon CloudWatch** | Gestión | Servicio de monitoreo y observabilidad para recursos y aplicaciones de AWS. |
| **AWS CloudFormation** | Gestión | Modelado y aprovisionamiento de recursos de infraestructura como código. |
| **AWS IAM** | Seguridad | Gestión de acceso e identidad para recursos de AWS de forma segura. |
| **AWS Auto Scaling** | Cómputo | Mantiene la disponibilidad de aplicaciones y permite escalar capacidad automáticamente. |
| **Amazon Route 53** | Redes | Servicio de DNS web escalable y de alta disponibilidad. |
